# Portfolio Website

A clean, responsive single-page portfolio built with **vanilla HTML, CSS, and JavaScript** — no frameworks, no build step. It showcases my background as a full-stack developer, my skills, and selected projects.

## ✨ Features

- Fully responsive layout (mobile → desktop)
- Smooth-scrolling navigation
- Dark, GitHub-inspired theme
- Lightweight and fast — a single `index.html` file
- Easy to deploy on GitHub Pages

## 🛠️ Built With

- HTML5
- CSS3 (Flexbox & Grid)
- Vanilla JavaScript

## 🚀 Getting Started

Clone the repo and open `index.html` in your browser:

```bash
git clone https://github.com/KhizarShah011/portfolio-website.git
cd portfolio-website
```

Then open `index.html`, or serve it locally:

```bash
python -m http.server 8000
```

Visit `http://localhost:8000`.

## 🌐 Deploy on GitHub Pages

1. Go to **Settings → Pages**
2. Set the source to the `main` branch
3. Your site will be live at `https://khizarshah011.github.io/portfolio-website/`
